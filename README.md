# Go 微服务网关配套前端代码

## 代码帮助

### 运行前端代码

- 切换到前端文件夹 `cd project_view`

- 确保本地环境安装了nodejs

```
node -v
v11.9.0
```

- 安装node依赖包

```
npm install
npm install -g cnpm --registry=https://registry.npm.taobao.org
cnpm install
```

- 运行前端项目

```
npm run dev
```
### 编辑器设置

- 用 vscode 打开 project_view 前端项目

- 安装格式化插件 ESLint、Vetur、vue-beautify
